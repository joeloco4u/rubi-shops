# Rubi.shops 🛍️

Tienda de ropa elegante para mujer - Demo Website

## Descripción

Página web de demostración para **Rubi.shops**, una tienda de ropa femenina elegante que incluye vestidos, accesorios y ropa para niñas.

## Estructura

```
rubi-shops/
├── index.html      # Estructura principal SPA
├── css/
│   └── styles.css  # Estilos completos
├── js/
│   └── script.js   # Funcionalidad SPA
├── assets/
│   └── images/     # Imágenes del proyecto
└── README.md       # Este archivo
```

## Características

- Diseño minimalista premium con acentos vibrantes
- Navegación SPA simulada (sin recarga)
- Totalmente responsive (mobile-first)
- Animaciones suaves y elegantes
- Tipografía bold moderna

## Tech Stack

- HTML5
- CSS3 (Flexbox + Grid)
- JavaScript Vanilla

## Ver proyecto

Abrir `index.html` en un navegador para visualizar el sitio.
